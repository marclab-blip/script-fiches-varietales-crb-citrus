# script-fiches-varietales-crb-citrus
Ce dépôt contient le script JavaScript utilisé dans le cadre du stage au CRB Citrus de San Giuliano pour automatiser la création de fiches variétales à partir de données Excel et d’images.

## Contenu

- generate_fiches_varietales.js : script principal de génération des fiches.
- README.md : description du fonctionnement général du script.

## Utilisation générale

Le script permet d’insérer automatiquement les données et les images associées aux variétés d’agrumes dans un modèle de fiche Excel.
